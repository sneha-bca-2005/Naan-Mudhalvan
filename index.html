Project Name: Digital Clock

About the Project: The Digital Clock is a simple web-based application that displays the current time in hours, minutes, and seconds. It updates in real-time and can be styled to match various themes or preferences. This project helps in understanding how to manipulate the DOM and work with time-based functions in JavaScript.

Features:

  1. Real-Time Display: Continuously updates to show the current time accurately.
  2. AM/PM Indicator: Displays time in a 12-hour format with AM/PM notation.

Customizable Design: Allows users to apply different styles, such as changing colors or fonts, to suit their preferences.

Technologies Used:
HTML: Structures the layout of the clock.
CSS: Styles the clock's appearance, including size, color, and positioning.
JavaScript: Handles the logic for fetching the current time and updating the display every second.
